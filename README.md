# 📊 Subscription Pricing Page

A responsive pricing page built using **HTML, CSS, Bootstrap, and jQuery**.
This project recreates a Bootstrap pricing example layout while adding interactive functionality.

---

## 🚀 Features

- Responsive layout using **Bootstrap grid system**
- Three pricing plans:
  - Free
  - Pro
  - Enterprise

- Interactive plan selection using **jQuery**
- Dynamic highlighting of selected plan
- Compare plans table
- Styled footer with branding
- Custom CSS to match Bootstrap example design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- jQuery

---

## 💡 Functionality

- Users can click on a **Subscribe button**
- The selected plan is highlighted
- Only one plan can be selected at a time
- Selected plan data (plan + price) is captured using jQuery
